Dúvidas Frequentes

Precisa instalar algo na minha infraestrutura?

Não. O sistema é 100% cloud (SaaS). Acesso via navegador. No plano Automation podemos conectar via API ou VPN.

O layout é atualizado automaticamente?

Sim. Quando o BACEN altera o leiaute (como fez com a IN BCB 622/2025), atualizamos o sistema sem custo adicional.

Quanto tempo leva a implementação do plano Automation?

Entre 15 a 45 dias, dependendo da complexidade dos sistemas internos.

E se eu errar algum dado na entrega?

O sistema valida todos os campos antes da geração. No plano Automation, a validação cruzada compara dados de múltiplas fontes.

Posso testar antes de assinar?

Sim. Oferecemos 15 dias de trial gratuito no plano Manual.
